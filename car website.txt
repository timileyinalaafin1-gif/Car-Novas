2.Explore page (Explore cars.html) Create a responsive car listings section for a car website.
The section must display three categories:
1. New Cars
2. Used Cars / Pre-Owned
3. Certified Pre-Owned
* Under each category, display a grid of car cards.
Each car card must include:
– A car image (use placeholder images)
– The name of the car
– The model year (example: 2022, 2023, etc.)
– A price tag (example: $28,995)
– Clean, modern card styling
* Layout Requirements:
– Use HTML5 and CSS3
– Make it fully responsive
– On desktop: show multiple cards in a row
– On mobile: cards should stack vertically
– Use CSS grid or flexbox for alignment
– Include a small title above each category
* Output should include:
– All HTML for the section
– All CSS needed for card styling and responsiveness
– Placeholder images for car photos
* Do not use JavaScript unless necessary.

3. Design Recommendations
                              * Layout: Use CSS Grid or Flexbox for responsiveness

                              * Colors: Dark background (#111 or #222) with light text (#f0f0f0 or #fff)

                              * Spacing: Include padding around the sections (50px top/bottom, 20px sides)

                              * Typography: Clear, legible font (Arial, Roboto, or similar)

                              * Links: Hover effect (e.g., color change or underline) to indicate interactivity

                              * Responsiveness: On smaller screens, sections should stack vertically
